🍽️ MyFoodbook (Recipe Finder App)
📌 Project Description

PantryPal is a simple web application that helps users discover recipes easily. Users can browse and view different recipes along with their ingredients and instructions. This project is built using basic web technologies and is designed for learning how to work with APIs.

🎯 Purpose

The main goal of this project is to:

Practice working with APIs using JavaScript
Understand how to fetch and display data dynamically
Build a clean and responsive user interface using HTML and CSS
🌐 API Used

This project uses the DummyJSON Recipes API:

👉 https://dummyjson.com/recipes

The API provides:

Recipe names
Ingredients
Cooking instructions
Images

✨ Features
Fetch and display recipes from API
Show recipe name, image, and details
Simple and clean UI
Dynamic content loading using JavaScript

🛠️ Technologies Used
HTML
CSS
JavaScript (Vanilla JS)
Fetch API
